Newbies

Young Rewired State foster a community of already-able young technical people. It is assumed that every person who attends one of our events can write and understand some form of programming or scripting language. However, it is completely inevitable that young people with less technical skill will come along to the centre during the week.

As a mentor, we suggest the following two activities to help aid less-skilled participants in learning the basics.

Intro to Web Development

To help newbies get started with the basics in HTML, CSS and Javascript, we have put together an

introductory exercise that can be found here:

http://www.yrs.io/get-started

The step-by-step format should allow for young people to work through the exercise without

constant supervision, giving you the freedom to move between groups.

Ruby’s Threads

Another exercise that can help when catering for skills gaps is our community project, Ruby’s

Threads:

http://rb.yrs.io

We have put all the code in our GitHub account ready for participants to fork and play around with.

This is a great exercise to introduce even your more advanced coders to GitHub:

https://github.com/youngrewired/rubys-threads

Participants will learn:

● How to setup git on your computer

● How to change 'branches' in git

● How to make changes to the project

● How to push your changes back up to GitHub

● How to submit a pull request on GitHub

We’d love to see a Ruby designed for each Festival of Code centre. Perhaps ‘designing a centre

Ruby’ is an exercise the whole centre can get involved with on day one, allowing for easier

integration of newbies.

Once your participants have completed the above two exercises and are feeling more confident

with code, it’s time to integrate them with the rest of the group.
